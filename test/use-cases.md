# FreePic 用户场景测试用例

## 测试环境

- URL: http://localhost:9000
- 测试账号: 138\***\*8001 / 138\*\***8002 / 138\*\*\*\*8003

## 1. 访客场景（未登录用户）

### UC-001: 浏览首页

**前置条件**: 用户未登录
**步骤**:

1. 访问首页
2. 查看 Hero 区域
3. 查看精选图片
4. 查看统计数据
5. 查看精选摄影师

**预期结果**:

- 页面正常加载
- 所有图片显示正确
- 统计数据正确显示

### UC-002: 搜索图片

**前置条件**: 用户未登录，在首页
**步骤**:

1. 在搜索框输入"风景"
2. 点击搜索按钮
3. 查看搜索结果

**预期结果**:

- 跳转到搜索结果页
- 显示相关图片
- 可以筛选和排序

### UC-003: 查看图片详情

**前置条件**: 用户未登录，在首页
**步骤**:

1. 点击图片
2. 查看大图
3. 查看EXIF信息
4. 查看相似图片

**预期结果**:

- 显示图片详情页
- 图片正常加载
- 显示相机型号、拍摄参数等信息

### UC-004: 访问上传页面（未登录）

**前置条件**: 用户未登录
**步骤**:

1. 点击导航栏"上传"按钮
2. 或直接访问 /upload

**预期结果**:

- 显示登录提示页面
- 显示"请先登录"信息
- 提供登录按钮

### UC-005: 访问用户主页

**前置条件**: 用户未登录
**步骤**:

1. 点击图片作者头像
2. 查看用户主页

**预期结果**:

- 显示用户信息和作品列表

## 2. 登录用户场景

### UC-006: 用户登录

**前置条件**: 用户未登录
**步骤**:

1. 点击"登录"按钮
2. 输入手机号
3. 获取验证码
4. 输入验证码
5. 点击登录

**预期结果**:

- 登录成功
- 跳转到首页
- 显示用户信息

### UC-007: 上传图片

**前置条件**: 用户已登录
**步骤**:

1. 点击"上传"按钮
2. 选择图片文件（JPG/PNG，<50MB）
3. 等待EXIF验证
4. 添加标签
5. 点击确认上传
6. 等待上传完成

**预期结果**:

- 显示图片预览
- 验证EXIF信息
- 上传成功提示
- 图片保存到用户账号

### UC-008: 上传图片（无EXIF）

**前置条件**: 用户已登录
**步骤**:

1. 选择无EXIF信息的图片
2. 查看验证结果

**预期结果**:

- 显示错误提示"该图片缺少EXIF信息"
- 无法上传

### UC-009: 查看个人主页

**前置条件**: 用户已登录
**步骤**:

1. 点击头像
2. 进入个人主页
3. 查看自己的作品
4. 查看统计数据

**预期结果**:

- 显示个人信息
- 列出所有上传的图片
- 显示获赞数、下载数等
